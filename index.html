Voice Assistant Project

Welcome to the Voice Assistant project! This voice-activated assistant uses Python libraries like pyttsx3, speech_recognition, and wikipedia to perform a variety of tasks based on your voice commands.

Project Features

  • Play music from your system
  • Fetch Wikipedia summaries on any topic
  • Open popular websites like YouTube, Google, and GitHub
  • Get your IP address
  • Tell jokes
  • Send emails with voice commands
  • And much more!

How to Use

Once downloaded, you can run the executable file and start giving commands to the assistant. It will listen to your commands and respond accordingly.
